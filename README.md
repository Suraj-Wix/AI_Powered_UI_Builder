# 🚀 AI Powered UI Builder

An AI-powered web application that generates beautiful, responsive UI components from simple text prompts. Built with **React, Vite, and Gemini AI**, this tool helps developers quickly create modern frontend designs without writing repetitive code.

![React](https://img.shields.io/badge/React-19-blue)
![Vite](https://img.shields.io/badge/Vite-Latest-purple)
![Gemini AI](https://img.shields.io/badge/Gemini-AI-orange)
![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Features

✅ Generate UI from natural language prompts

✅ Multiple framework support

* HTML + CSS
* HTML + Tailwind CSS
* HTML + Bootstrap
* HTML + CSS + JavaScript
* HTML + Tailwind + Bootstrap

✅ AI-powered code generation using Gemini AI

✅ Live code editor with Monaco Editor

✅ One-click code copy

✅ Responsive and modern UI designs

✅ Fast and lightweight React application

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/AI-Powered-UI-Builder.git
cd AI-Powered-UI-Builder
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Create Environment Variables

Create a `.env` file in the project root:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

Application will run on:

```text
http://localhost:5173
```

## 📸 Workflow

1. Enter your UI description.
2. Select desired framework.
3. Click Generate.
4. Gemini AI creates the UI code.
5. Review in Monaco Editor.
6. Copy and use instantly.
